# ReelPull SaaS 計畫書

## 產品定位

**簡單、快速的 Instagram Reels 下載工具**

核心功能就兩個：
1. 貼上連結 → 下載影片
2. 輸入帳號 → 下載該帳號所有 Reels

不做多餘的東西，專注把這兩件事做到極致。

---

## 功能規劃

### 免費版
- 單一影片下載（無限制）
- 基本影片管理與重命名

### Pro 版（$5/月 或 $30/年）
- **帳號批量下載**：輸入 IG 帳號，一鍵下載所有 Reels
- **Hashtag 下載**：輸入 #tag，下載熱門 Reels
- 更快的下載速度
- 無廣告

就這樣，簡單明瞭。

---

## 帳號爬蟲功能（核心開發）

### 使用流程

```
1. 用戶輸入帳號名稱（如 nike）
2. 顯示該帳號資訊（頭像、粉絲數、Reels 數量）
3. 用戶點擊「全部下載」
4. 背景批量下載，顯示進度
5. 完成後可統一管理/重命名
```

### 技術實作

```
方案 A：無需登入（公開帳號）
- 直接爬取 Instagram 網頁版
- 解析 JSON 資料取得 Reels 列表
- 優點：簡單、無帳號風險
- 缺點：只能抓公開帳號

方案 B：需要登入（私人帳號）
- 用戶提供自己的 IG Cookie
- 可存取私人帳號
- 優點：功能更完整
- 缺點：有帳號風險、維護成本高

建議：先做方案 A，驗證市場後再考慮 B
```

### 反爬蟲對策

```
1. 請求頻率控制
   - 每次請求間隔 2-5 秒（隨機）
   - 避免短時間大量請求

2. 代理 IP 輪換
   - 使用代理池
   - 被封就換 IP

3. User-Agent 輪換
   - 模擬不同瀏覽器

4. 降級策略
   - 被擋時自動降速
   - 通知用戶稍後再試
```

## 網站優化方向

### UI/UX 改進

**首頁**
- 超大輸入框，一眼就知道怎麼用
- 輸入連結或帳號名稱都可以
- 自動辨識輸入類型

**下載體驗**
- 拖放支援（直接拖連結進來）
- 下載進度即時顯示
- 批量下載時顯示總進度

**影片管理頁**
- 縮圖預覽更大更清楚
- 批量選擇、批量重命名
- 一鍵全部下載到本機

### 效能優化

- 下載速度更快（多線程）
- 頁面載入更快
- 支援斷點續傳
